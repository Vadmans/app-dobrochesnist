# Доброчесність — бекенд (API + база)

Один сервіс для обох клієнтів: застосунку користувача й адмін-панелі.
За замовчуванням працює на SQLite — нічого окремо ставити не треба.

## Запуск

```bash
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Після запуску:
- **http://localhost:8000/docs** — інтерактивна документація, де можна
  створювати/редагувати події прямо з браузера (зручно для перевірки).
- **http://localhost:8000/events** — список подій (JSON), який тягне застосунок.

При першому запуску база автоматично наповнюється чотирма демо-подіями.
Файл бази — `dobrochesnist.db` — з'явиться поруч.

## Ендпоінти

Для застосунку користувача (читання):
- `GET /events` — усі події
- `GET /events/{id}` — одна подія
- `POST /events/{id}/view` — зафіксувати перегляд (для статистики)

Для адмін-панелі (керування):
- `POST /events` — створити
- `PUT /events/{id}` — змінити
- `DELETE /events/{id}` — видалити
- `GET /stats` — статистика переглядів

## Перехід на PostgreSQL (для продакшену)

1. Запустіть PostgreSQL і створіть базу `dobrochesnist`.
2. Задайте змінну середовища перед запуском:
   ```bash
   export DATABASE_URL="postgresql+psycopg://user:pass@localhost:5432/dobrochesnist"
   pip install "psycopg[binary]"
   ```
   Решта коду не змінюється — SQLAlchemy сам працює з обома базами.

## Що додати далі (етап 2)

- **Автентифікація адміна** — захистити `POST/PUT/DELETE` перевіркою токена,
  щоб керувати подіями могла лише відповідальна особа.
- **Розсилка нагадувань** — фонове завдання, що раз на день шукає події,
  до строку яких лишилось 30/10/3/0 днів, і шле email/push.
